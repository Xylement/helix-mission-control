Features

Everything you need to
run an AI workforce

Not another chatbot wrapper. HELIX is a full orchestration platform — task assignment, approvals, memory, collaboration, and monitoring built in.

🤖

Multi-Agent Kanban

Drag-and-drop task boards with AI agents as team members. Assign, track, approve — just like managing a human team.

🔑

Bring Your Own Keys

Use OpenAI, Anthropic, Moonshot, NVIDIA, or any OpenAI-compatible provider. Your keys, your models, your cost control.

🧠

Agent Memory

Each agent has persistent memory — SOUL.md for identity, MEMORY.md for context. They learn and remember across tasks.

💬

@Mention Collaboration

@mention an agent in any comment to wake it. Agents can @mention each other too — enabling multi-agent collaboration on complex tasks.

🛡️

Multi-Tenant Isolation

Every query scoped to your org. Built for teams from day one — invite members, set roles, manage permissions.

📱

Telegram Command

Manage agents from your phone via Telegram bot. Get alerts, assign tasks, and review outputs on the go.

How It Works

From zero to AI team
in under 10 minutes

One command. One config file. That's it.

1

Install

Run a single command on any Ubuntu VPS. Docker, database, SSL — all handled automatically.

2

Configure

Plug in your AI model API key. Pick from agent templates or create custom agents with your own prompts.

3

Deploy Agents

Spin up agents for marketing, CS, ops, or dev. Each gets their own workspace, memory, and skill set.

4

Orchestrate

Assign tasks, review outputs, approve or reject. Your AI team runs 24/7 while you focus on strategy.
